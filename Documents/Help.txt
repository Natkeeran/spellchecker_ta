1. Opening the file.

	This option select the file for Spell check from the local disk. Any text file  with TAB font can spell check using this software.

2. Spell Check option.

	This option check the word in the document is correct or not. If a word is error means, that word is highlighted.

3. Getting the suggestion

To get the suggestion for the erraneous word do the following steps.

 *  Place the cursor in erraneous word then click the right mouse button. 
 *  You will get the list of suggestions for the erraneous word. 

4. Place the suggestion in the document

 To change the suggestion into document do the following steps.

 *  Select a suggestion from the list and click the change button. 
 *  Correct word will change in the document instead of error word.  

5. Leave the suggestion

	If U don't want to select the suggestion just click the button "leave it". So no changes will appear in the document.

6. Add the word into ditionary

	Option to add a word to the dictionary is also provided after which any occurrence of that particular word or its morphological variants will not result in a error. Select the word and click the add button to add that word into dictionary.

7. Cancel the document

	Using this option you can cancel the current document from the editable window.
