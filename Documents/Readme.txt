ABOUT
	Tamil Spell Checker is a tool used to check the spelling of Tamil words. It provides possible suggestions for wrong words. An important feature is the use of Morphological Analyzer for suggestion generation.

REQUIREMENTS
	 * J2RE 1.4
	 * Operating System: Windows/Linux 
	 * Any Tamil Keyboard driver to type Tamil Text
	 * Any TAB Font should be installed in the System to manipulate Tamil text

HOW TO RUN
	Double click the batch file SpellChecker.bat
	OR
	* At command prompt, go to the directory where you copied and type the command "java SpellChecker"
